# GlowGuide – Skincare Recommendation System

GlowGuide is a personalized skincare recommendation system that suggests suitable products based on user inputs such as skin concern, skin type, lifestyle, and budget.

## Features

* Personalized product recommendations
* Rule-based scoring system
* Lifestyle-based suggestions
* AI-style skin report
* Budget filtering

## Tech Stack

* Python
* Streamlit
* Pandas

## How It Works

The system takes user inputs, filters products from a dataset, applies a scoring algorithm, and displays the most relevant recommendations along with basic lifestyle advice.

## Run the Project

pip install -r requirements.txt
streamlit run app.py

## Project Type

Rule-based recommendation system developed as part of a laboratory project.
